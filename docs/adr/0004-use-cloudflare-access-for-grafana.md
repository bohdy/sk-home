# Use Cloudflare Access for Grafana

Grafana will use `grafana.bohdal.name` as its canonical hostname. Internal split DNS will resolve the name directly to Grafana's Cilium LoadBalancer VIP so observability remains available during internet or Cloudflare outages, while public DNS will route the same name through a shared cluster Cloudflare Tunnel protected by Cloudflare Access. Grafana authentication remains enabled on both paths; Cloudflare Access uses Google identity, exact-account allowlisting, and MFA as an additional external perimeter rather than the application's sole authentication layer.
