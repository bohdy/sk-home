# Adopt VictoriaMetrics for metrics

Home Infrastructure Observability will use VictoriaMetrics as the v1 metrics backend, deployed through the official `victoria-metrics-k8s-stack` Helm chart with VictoriaMetrics Operator, `VMSingle`, `VMAgent`, `VMAlert`, and `VMRule` resources. The single-node store will retain one year of raw metrics without simulated downsampling; actual ingestion will be measured before another storage tier or VictoriaMetrics edition is considered. This keeps Grafana and Prometheus-compatible exporters as the integration contract while avoiding the heavier Prometheus/Mimir path for a single home cluster.
